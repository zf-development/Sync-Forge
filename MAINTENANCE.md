# Maintenance Policy

This document outlines the maintenance policy for the Sync mod.

## Maintenance Status

**Current Status**: âœ… Actively Maintained

## Maintenance Schedule

- **Regular Updates**: As needed for bug fixes and compatibility
- **Version Updates**: When new Minecraft/NeoForge versions are released
- **Response Time**: We aim to respond to critical issues within 48 hours

## Supported Versions

### Currently Supported
- **Minecraft 1.20.1** with NeoForge 47.x

### Planned Support
- Future Minecraft versions will be evaluated on a case-by-case basis

## Version Support Lifecycle

1. **Active Support**: Latest version receives bug fixes and new features
2. **Security Updates**: Critical security issues will be patched for recent versions
3. **End of Life**: Versions older than 2 major Minecraft versions will no longer receive updates

## Contributing

We welcome contributions! Please see [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

### Areas Where Help is Needed
- Bug fixes
- Performance improvements
- Documentation
- Translations
- Testing on different mod configurations

## Reporting Issues

Please use GitHub Issues to report bugs or request features. Include:
- Minecraft version
- Mod version
- NeoForge version
- Steps to reproduce
- Crash logs (if applicable)
- List of other mods installed

## Feature Requests

Feature requests are welcome but not guaranteed to be implemented. Priority is given to:
1. Bug fixes
2. Compatibility updates
3. Performance improvements
4. Quality of life improvements

## Breaking Changes

We will try to minimize breaking changes between versions. When breaking changes are necessary:
- They will be documented in the CHANGELOG.md
- Migration guides will be provided when possible
- Advance notice will be given for major changes

## Long-term Maintenance

If this project becomes unmaintained:
- An announcement will be made in the repository
- The repository will be marked as archived
- Community forks are welcome to continue development

## Contact

For maintenance-related questions:
- Open an issue on GitHub
- Check existing issues and discussions
- Review the documentation
